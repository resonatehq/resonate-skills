# Resonate Patterns

A catalog of patterns for building durable functions with Resonate.

Each pattern follows a standard format:

- **Names**: Names for this pattern
- **Problem**: When to use this pattern
- **Solution**: How to implement this pattern
- **Variants**: Common variations
- **Related Patterns**: Connections to other patterns

## Patterns

| Pattern | Description |
|---------|-------------|
| [Wait for External Event](./wait-for-external-event.md) | Pause execution until an external actor (human, agent, service, webhook) signals completion |
